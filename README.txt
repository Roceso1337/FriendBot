FriendBot by Dan Sze, Robert Carniero, Albert Shin, Brendan Courson

=== INSTRUCTIONS ===

1. Install all required Python packages.
> pip install -r requirements.txt

2. If you haven't already, import nltk and run nltk.download()
> python
>>> import nltk
>>> nltk.download()

Download all the packages.

3. Run python app.py
> python app.py

4. Go to localhost:5000 using your favorite Internet Browser
